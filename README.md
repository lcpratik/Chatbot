# 💬 Chatbot

A lightweight, floating chatbot widget built with vanilla JavaScript, HTML, and CSS — no frameworks, no dependencies. Features a clean floating icon UI that expands into a fully functional chat interface.

---

## Features

- 💬 Floating chat icon that toggles the chatbot open/closed
- 🧠 Interactive message input and response rendering
- 🎨 Fully custom styled UI with smooth transitions
- ⚡ Zero dependencies — pure HTML, CSS, and JavaScript
- 📱 Responsive layout that works across screen sizes

---

## Tech Stack

| Layer     | Technology              |
|-----------|-------------------------|
| Structure | HTML5                   |
| Styling   | CSS3                    |
| Logic     | Vanilla JavaScript (ES6)|
